Etiquetas Básicas de HTML

Etiqueta Descripción
HTML indica el comienzo del documento HTML.
head indica que empieza la cabecera de la página.
title Título que tendrá la página
body es el cuerpo de la página, aqui pondremos todo lo que queremos que cargue la página.
h1,h2... Títulos o encabezados.
a Para insertar un enlace
table es una tabla, dentro de ella insertamos filas con tr y celdas con td.
p Para insertar un párrafo.
br Para insertar un salto de línea.
img Para insertar una imagen.
ul Para insertar una lista desordenada, con la etiqueta li insertamos los datos que tendrá.
ol Para insertar una lista ordenada., con la etiqueta li insertamos los datos que tendrá.
b/strong Para hacer que el texto se vea en negritas
u Para hacer que el texto aparezca subrayado
i/em Para hacer que el texto este en cursiva
